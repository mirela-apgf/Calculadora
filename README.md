🧮 Calculator App

Mobile application developed using Java and XML for performing mathematical calculations quickly and efficiently.
The app provides a simple and intuitive interface, allowing users to perform everyday calculations with ease.

<br>
🚀 Features

- Basic operations (addition, subtraction, multiplication, division)
- Real-time calculation display
- Clean and intuitive UI


<br>
🛠 Technologies

- Java
- XML (Android Layouts)
- Android Studio

<br>
💡 Usability

The application was designed with a focus on simplicity and usability.
Users can perform calculations quickly through an intuitive interface, making it practical for daily use.
